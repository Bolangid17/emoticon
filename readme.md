# emoticon [![Build Status][travis-badge]][travis] [![Coverage Status][codecov-badge]][codecov]

Info on ASCII emoticons.  :p

## Installation

[npm][]:

```bash
npm install emoticon
```

## Usage

```javascript
var emoticon = require('emoticon');

console.log(emoticon.slice(0, 3));
```
